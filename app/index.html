关于我们

IM Web Conf 2017，是一场专注于web前端领域的开发者大会。

历经5届的成功举办，第6届IM Web Conf将于2017年9月9日在深圳召开。届时，大会将邀请国内外的知名讲师， 为大家分享前端技术方方面面的经验。大会为前端开发者提供了交流与学习的机会，一起推动国内外前端技术的 发展，深入挖掘前端技术可以创造的价值。

欢迎大家齐聚一堂，以技会友，共同分享爱那个技术的乐趣。
